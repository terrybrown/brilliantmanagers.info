Theme toggle component (header, top-right)

🌙 Dark ☀️ Light Implemented via next-themes + ThemeProvider. Persists to localStorage, respects prefers-color-scheme on first visit. A single ThemeToggle component lives in the nav bar.

Implementation note: Tailwind v4 dark mode is via the dark class on <html>, controlled by next-themes. shadcn/ui uses CSS custom properties that already switch with the .dark class — so all shadcn components get dark mode for free. Custom components use dark: Tailwind variants. Default respects system preference on first visit.
